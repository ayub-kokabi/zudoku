## خوش آمدید به زودوکو

Zudoku یک تولید کننده مستندات است که به شما اجازه می دهد مستندات زیبا را به راحتی ایجاد کنید. از
MDX، OpenAPI پشتیبانی می کند و کاملا قابل سفارشی است.

### شروع به سفارشی سازی

می توانید سایت مستندات خود را با تغییر فایل `zudoku.config.js` در ریشه پروژه خود سفارشی کنید. چرا
رنگ اصلی سایت خود را تغییر ندهید؟

```json
{
  // ...
  "theme": {
    "light": {
      "primary": "316 100% 50%",
      "primaryForeground": "360 100% 100%"
    },
    "dark": {
      "primary": "316 100% 50%",
      "primaryForeground": "360 100% 100%"
    }
  }
  // ...
}
```
